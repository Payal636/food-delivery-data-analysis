# Food Delivery Data Analysis

This project analyzes food delivery data using CSV, JSON, and SQL sources.
The datasets are merged using Pandas and analyzed to answer business and
analytical questions related to revenue, users, cuisine, and ratings.

# Files
- orders.csv – Transactional data
- users.json – User master data
- restaurants.sql – Restaurant master data
- food_delivery_analysis.ipynb – Jupyter notebook analysis
- final_food_delivery_dataset.csv – Final merged dataset
